# 用户权限流程
## 表示用户账户状态的枚举
```java
	public enum AccountStatus {
		UNVERIFIED, // 用户只注册，未经邮箱验证,拥有浏览权限
		ID_VERIFIED,// 验证了学校邮箱，可以正常交易
		REAL_NAME_VERIFIED,// 完成实名认证，拥有最高权限
	}
```

## 重设密码与忘了密码
重设密码直接就登陆之后修改
忘了密码，就给邮箱发送密码。

## 用户权限
* 未认证用户，可以正常浏览，搜索，无法下单，会有未认证标志。
* 学生认证用户就可以正常交易，发布书籍，会有认证标志。
